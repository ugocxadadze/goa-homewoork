ჩემს შესახებ

გამარჯობა! მე ვარ უგო ავტანდილი, ვნებიანი ვებ დეველოპერი, ფრონტ-ენდის ტექნოლოგიების ძლიერი ფონზე. მე მსიამოვნებს საპასუხო და მოსახერხებელი ვებსაიტების შექმნა. დიზაინის მახვილი თვალით და კოდის ხარისხისადმი ერთგულებით, მე ვცდილობ მივაწოდო განსაკუთრებული ვებ გამოცდილება.

უნარები

მე სპეციალიზირებული ვარ HTML, CSS და JavaScript-ში. გარდა ამისა, მაქვს გამოცდილება React-თან, Node.js-თან და სხვა ვებ ტექნოლოგიებთან. მე ასევე ვფლობ ვერსიის კონტროლს Git-ით და კარგად მესმის საპასუხო დიზაინის პრინციპები.

პროექტები

პროექტი პირველი

პირველი პროექტის აღწერა. ეს პროექტი მოიცავდა HTML, CSS და JavaScript-ის გამოყენებას საპასუხო ვებსაიტის შესაქმნელად. მიზანი იყო ბრაუზერის ჯვარედინი თავსებადობისა და მომხმარებლის გლუვი გამოცდილების უზრუნველყოფა.

პროექტი მეორე

მეორე პროექტის აღწერა. ეს იყო უფრო რთული პროექტი, რომელშიც მონაწილეობდნენ React და Node.js. იგი მოიცავდა სრული დასტას აპლიკაციის შექმნას მასშტაბურ და შენარჩუნებულ კოდზე ფოკუსირებით.
